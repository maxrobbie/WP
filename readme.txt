== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `wp-event-plugin` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. After plugin installation admin can import json data with help of setting interface. Event > Setting.
4. User can view upcoming event list on event archive page : site_url/events
5. Export event data in json format url is  : site_url/wp-json/wp/v2/event/
